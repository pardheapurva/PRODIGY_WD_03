## Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript (DOM Manipulation)  

##  How It Works

- The game maintains a `gameState` array of 9 positions.
- Winning combinations are checked after every move.
- AI selects a random empty cell.
- Game stops when:
  - A player wins
  - The board is full (Draw)
